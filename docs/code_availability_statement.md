# Draft Code Availability Statement

RNA-seq and mass-spectrometry data have been deposited at GEO and will be publicly available as of the date of publication. Accession numbers will be listed in the key resources table. The MAMMOTH Web Portal can be accessed at https://jannisbuecking-mamoth.share.connect.posit.cloud/.

The MAMMOTH web portal source code and packaged app data will be publicly available at https://github.com/jannisbuecking/MAMMOTH as of the date of publication. The custom multi-omics analysis code will be publicly available at https://github.com/jannisbuecking/Buecking_et_al_2026_MAGEL2_multi-omics as of the date of publication. Any additional information required to re-analyze the data reported in this paper is available from the lead contact upon request.
